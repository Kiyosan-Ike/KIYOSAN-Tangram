# KIYOSAN-Tangram

## できること
- iPadのホーム画面からアプリのように起動
- Safariのアドレスバーを表示せずに使用
- 一度読み込めばオフラインで使用
- クリアした問題の星印を端末内に保存

## 大切な点
PWAのインストールとオフライン機能は、`file://` で直接開いた場合には働きません。
HTTPS対応のWebサーバーに、このフォルダの中身をそのまま置いてください。

## iPadへの追加方法
1. Safariで公開した `index.html` を開く
2. 共有ボタンを押す
3. 「ホーム画面に追加」を選ぶ
4. 「KIYOSAN-Tangram」を追加する

ホーム画面の専用アイコンから開くと、Safariのアドレスバーなしで起動します。

## 星の記録
星印はブラウザの `localStorage` に保存されます。
同じiPad・同じブラウザ・同じ公開URLで使う限り、記録は残ります。
「星をリセット」ボタンを押すと消去できます。
